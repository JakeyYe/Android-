## Java File类详解

1，File对象也可以表示不存在的文件的，其实File对象只是代表一个抽象路径，不是代表文件对象，构建一个File类的实例并不会在机器上创建一个文件，不管文件是否存在的可以创建任意文件名的File实例，可以调用File实例的exists()方法判断文件或目录是否存在；

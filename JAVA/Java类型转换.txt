
在Java中强制类型转换分为 基本数据类型转换 和 引用数据类型转换

   基本数据类型的转换，分为自动转换和强制转换。自动转换是程序在执行过程中“悄然”进行的转换，不需要用户提前声明，一般是从位数低的类型向位数高的类型转换；强制类型转换则必须在代码中声明，转换顺序不受限制。

自动基本数据类型转换
自动转换按从低到高的顺序转换。不同类型数据间的优先关系如下：
     低--------------------------------------------->高
     byte,short,char-> int -> long -> float -> double


强制基本数据类型转换
强制转换的格式是在需要转型的数据前加上“( )”，然后在括号内加入需要转化的数据类型。有的数据经过转型运算后，精度会丢失，而有的会更加精确。

类的向上转型和向下转型
在进行对象的向下转型前，必须首先发生对象向上转型，否则将出现对象转换异常。

编译器在编译时只会检查类型之间的是否存在继承关系，有则通过；而在运行时就会检查它的真实类型，是则通过，否则抛出ClassCastException异常。
